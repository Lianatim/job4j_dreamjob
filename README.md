# Проект "Работа мечты"

## О проекте:

Web-приложение для размещения  вакансий и резюме.

## Стек технологий

- **Сборка приложения в Maven 3.8**
- **Java 18**
- **Spring Boot 2.7.3**
- **HTML 5**
- **Thymeleaf 2.7.3**
- **BOOTSTRAP 4.4.1**
- **Mockito 4.8.0**
- **H2**

## Требования к окружению

- **Java 18**
- **Maven 3.8**
- **PostgresSQL 14**

## Запуск проекта

- **Создать БД**
``` shell 
create database dreamjob;
```
- **Запустить проект по команде**
``` shell 
mvn spring-boot:run
```
